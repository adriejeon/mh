디자인시스템

Year
2022.11 - 2023.04
Role
Product designer
Scope
가이드라인 작성, 반응형 웹 디자인, 프론트 웹 구축
Tools

Overview

디자인시스템은 디자인 원칙과 규격, 재사용이 가능한 UI 패턴과 컴포넌트, 코드를 포괄하는 시스템입니다. 확장된 디자인시스템 개념에는 브랜드 원칙과 UX 원칙을 포함하며 현재 다누시스의 디자인시스템 단계는 다누시스 제품에서 사용할 UX/UI 컴포넌트와 패턴까지 제작되었습니다. 미리 정립한 디자인 패턴에 맞춰 제품 기능의 UI를 사용할 수 있으며 그로 인해 유저의 사용성을 개선할 수 있고 작업 소요 시간을 단축할 수 있습니다. 또한 디자인 업무 히스토리를 공유하지 않은 여러 사람이 디자인에 참여하여도 디자인 일관성을 확보할 수 있어 히스토리 운용에도 유용합니다.

기존의 문제점 작업 프로세스 솔루션 결과 기대효과

기존의 문제점

(1) 일관되지 않은 디자인

일관되지 않은 UI가 사용성을 떨어뜨리고 있었습니다. 일반의 버튼 상태와 마우스 오버했을 경우의 버튼 상태, 눌렀을 때의 버튼 상태 값에 대한 규칙이 부재인 상태였습니다. 이런 사소한 부분들이 쌓여서 사용자의 사용성을 저하시키는 주요한 디자인적 문제로 작용하고 있었습니다.

(2) 비효율적인 작업의 지속적인 발생

사이트 마다 새로운 IoT 디자인을 제작하면서 같은 기능임에도 다시 디자인 해야만 했고 그로 인해 비효율적인 작업이 발생할 수 밖에 없었습니다. 버튼 스타일이 바뀌거거나 새로운 텍스트 인풋 창이 추가되는 경우에는 일일이 스타일을 다시 잡아야 했고, 퍼블리싱에서도 마찬가지로 새로운 코드 규칙을 짜야 했습니다.

(3) 디자인 확장성 부족

디자인 컴포넌트 끼리 연결되지 않아 요소를 확장하거나 고도화 하는 데 어려움이 있었습니다. 이전의 디자인을 재활용하여 촉박한 마감일을 좇다 보니 사용성을 위해 디자인 개선과 문제 해결을 위해 고민할 수 있는 시간을 갖을 수 없었습니다.

작업 프로세스

디자인의 최초 뼈대가 되는 색깔, 글자, 아이콘 등으로 구성되는 파운데이션을 선행하여 작업하였습니다. 그 다음으로 파운데이션을 조합해서 구성되는 버튼, 뱃지, 드롭다운 등의 컴포넌트를 제작하고 마지막으로 컴포넌트를 조합해서 구성되는 테이블, 버튼 그룹 등의 패턴을 완성하였습니다.

솔루션 결과

(1) UI 디자인의 분업화

상세한 디자인 가이드 규칙과 사내 작업 툴에 구축해 놓은 컴포넌트로 인해 UI 디자인이 디자이너만의 업무에 국한되지 않고 유관 팀 구성원들도 UI 디자인에 참여할 수 있게 되었습니다. 기존의 디자이너만 작업할 수 있던 중앙 집중화 모델에서 구성원 모두가 참여할 수 있는 분업화 모델로 확장할 수 있었습니다.

(2) 공동의 디자인 자산 확충

UI 디자인 공동의 자산을 쌓았고 실제 업무에 사용하며 내용을 보충하고 보완하고 있습니다. 디자인 톤앤매너 유지를 위해 별도의 미팅을 진행하지 않아도 모두 ‘어떤 컴포넌트를 어디에 사용하면 되는지’ 알고 있고 그로 인해 공동의 지적 자산을 확보하였습니다.

(3) 커뮤니케이션 기준 마련

협업 팀 마다 문화적 차이, 업무의 차이, 시야의 차이로 인해 커뮤니케이션이 어려울 수 있고 그로 인해 생기는 미스커뮤니케이션은 업무 작업 소요 시간을 늘릴 수 있습니다. 디자인시스템의 ‘토큰’ 개념을 통해 컴포넌트 마다 ‘이름 명명’을 하였고, 그로 인해 미스커뮤니케이션으로 인해 소요되는 불필요한 시간을 줄일 수 있었습니다.

기대효과

(1) 일관된 브랜드 정체성과 높은 사용성 확보

다양한 사이트에 걸쳐 일관된 인터페이스 디자인을 전달하고 다누시스만의 UX 구조를 확보하고 개선해 나가며 사용자와의 커뮤니케이션 서비스에 주목한다면 ‘락인 효과’를 기대할 수 있을 것 입니다. 이렇게 확보된 고객은 타 사와의 경쟁에서 우위를 선점할 수 있게 해주는 열쇄가 될 것 입니다.

(2) 웹과 앱에 동일한 환경 제공

웹과 앱 디바이스 사이의 디자인 간극을 없앨 수 있습니다. 웹에 사용한 컴포넌트와 동일하게 앱을 제작하거나 칼럼 시스템(참고자료)을 활용하여 반응형 플랫폼 제작을 드는 작업 소요 시간을 줄이고 효율적이고 일관적으로 디자인 완성도를 높일 수 있습니다.
